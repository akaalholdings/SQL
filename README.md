# SQL

This repository currently contains one standalone tool:

- `azsql-BulkCopy/`
  Chunked Azure SQL Database to Azure SQL Database bulk copy tool for large table seed loads.

## Quick Start

```bash
cd azsql-BulkCopy
cp .env.example .env
python azure_sql_bulkcopy.py
```

See [azsql-BulkCopy/README.md](/Users/balwinder/Development/SQL/azsql-BulkCopy/README.md) for configuration and tuning guidance.
